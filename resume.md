## 프로젝트

### C&C인터내셔널 AI Labs 인턴 지원 프로젝트 *(자체 개발)*
`2026.04` | Claude, GPT, Python, Streamlit

C&C인터내셔널 지원을 위해 뷰티 ODM 산업을 자체 분석하고, 실제 업무에 적용 가능한 AI 자동화 MVP 2종을 직접 개발하여 제출.

- **규제 성분 검증 시스템**: 화장품 원료 성분 리스트를 입력하면 EU·미국·한국 규제 데이터베이스 기준으로 금지·제한 성분을 자동 플래그하는 MVP 개발 (Claude API + Streamlit)
- **RFQ 자동 파서**: 고객사로부터 수신되는 비정형 RFQ(견적 요청서) 문서를 파싱하여 표준 양식으로 구조화하는 자동화 파이프라인 구현 (GPT API + Python)
- **산업 분석**: ODM 비즈니스 모델, SAP 도입 이후 운영 과제, AI 적용 가능 영역(성분 검증·RFQ 처리·재고 예측)을 분석하여 문서화
